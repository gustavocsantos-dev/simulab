🧠 Sobre o Projeto

Este projeto foi desenvolvido com o objetivo de me auxiliar nos estudos durante o curso de Análise e Desenvolvimento de Sistemas (ADS). A ideia surgiu da necessidade de revisar e fixar melhor o conteúdo das matérias por meio de simulados personalizados, criados por mim mesmo.

Inicialmente, o código foi elaborado em Python, como uma forma prática de aplicar o que eu aprendia na faculdade, unindo lógica de programação, estrutura de dados e pensamento computacional. Com o tempo, o projeto evoluiu para uma aplicação web interativa, que permite criar, executar e gerenciar simulados de forma intuitiva e acessível. Além de reforçar meus conhecimentos técnicos, este projeto representa também minha jornada de aprendizado e evolução como desenvolvedor.
